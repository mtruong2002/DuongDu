Những Khóa học dành cho bạn

Html cơ bản

HTML là một ngôn ngữ đánh dấu được thiết kế ra để tạo nên các trang web trên Word Wide Web. Nó có thể được trợ giúp bởi các công nghệ như CSS và các ngôn ngữ kịch bản giống như JavaScript.

CSS cơ bản

CSS là ngôn ngữ định dạng và tạo phong cách cho trang web. Bạn có thể hiểu đơn giản, nếu HTML đóng vai trò tạo nên bộ khung cho trang web thì CSS sẽ là phần bên ngoài hiển thị cho người dùng.

JavaScript Cơ Bản

Javascript là một ngôn ngữ lập trình kịch bản dựa vào các đối tượng có sẵn hoặc do người dùng tự định nghĩa.
